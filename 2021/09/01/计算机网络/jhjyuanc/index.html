第一章 二层交换机（神州数码）

SSH

SW(Config)#interface vlan 1
​SW(Config-If-Vlan1)#ip address 192.168.221.199 255.255.255.0 //配置管理地址

SW(Config-If-Vlan1)#no shutdown
​SW(Config-If-Vlan1)#exit
​SW(Config)#ssh-service enable //启用SSH
​SW(Config)#username LJQ password 0 LJQ //配置用户和密码
​SW(Config)#radius-server authentication host 192.168.221.199 //配置计费主机
SW(config)#aaa enable //启用AAA认证

Telnet

二层交换机默认开启Telnet

启动：
    SW(Config)#telnet-server enable                                  //启动Telnet
    SW(Config)#username LJQ password 0 LJQ                        
    
关闭：SW(Config)#no telnet-server enable

delete startup-config
reboot

第三章 PVLAN(私有vlan)

Vlan 100
Private-vlan primary //设置主vlan
Vlan 200
Private-vlan community //设置群体vlan
Vlan 300
Private-vlan isolated //设置隔离vlan
Vlan 100
Private-vlan association 200,300

注：
群体vlan代表在群体vlan中的主机都可以进行相互通讯，且通过vlan100 出去
隔离vlan代表在隔离vlan中的主机无法进行相互之间的通讯，同时各个主机也通过vlan100出去
